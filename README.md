# Mood Tracker Web App

A simple web app to track your daily mood with optional notes. Stores your mood history in the browser using ```localStorage```.

- HTML
- CSS
- JavaScript (Vanilla)
- localStorage

## How to Use
1. Open `index.html` in your browser
2. Select a mood
3. Optionally write a note explaining your mood
4. Click "Save Mood"
5. Your mood history will appear below, showing the date, mood and note
